# Haute Couture App

This project was built using **Express.js** and **Node.js**.

---

## Available Scripts

In the project directory, you can run:

### `npm install`

Installs all the dependencies required for the project.  
Make sure to run this command before starting the server for the first time.

---

### `npx nodemon server.js`

Runs the app in development mode with **auto-reload** on file changes.  
The server will restart automatically whenever you save a file.

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

---

## Notes

- To stop the server, press **Ctrl + C** in the terminal.
- If you prefer to start the server once (without auto-reload), you can also use:
  ```bash
  node server.js
